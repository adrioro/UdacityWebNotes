ABOUT THE WEB

The web is composed of HTML (Hypertext Markup Language) documents. The browsers use this language to interpret web pages. The web includes the client (our computer and browser), the Internet, and the servers. The client and servers interact through the HTTP (Hypertext Transfer Protocol).


WHAT ARE HTML ELEMENTS?

HTML tell the browser how to display the content. For example:
span style="color:black;font-weight:normal"> The <b></b> tag makes the 'text' content bold.
The <em></em> tag makes the 'text' content italic.


ATTRIBUTES

<tag attr="value"> content </tag>

THE "TREE LIKE" STRUCTURE

Html is organized in a tree like structure that makes it easier for us to detect where the elements are, and how they relate to each other. Elements within the document are displayed inside one another depending of their position in the page. The elements relate much like family members in a family tree do. In fact, they are nested boxes, one inside another. Depending of how deeply each element is nested within the page, its indentation would be within the DOM (document object model).
